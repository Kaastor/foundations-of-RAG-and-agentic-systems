# Summary
After firmware 3.2, the installation torque for the V14 sensor changes from 0.42 N-m to 0.48 N-m.

# Control
SB-118 supersedes SB-104 and the torque paragraph in the 2024 installation manual.

# Evidence table
| Product | Old torque | New torque | Applies after |
| --- | --- | --- | --- |
| V14 sensor | 0.42 N-m | 0.48 N-m | firmware 3.2 |

# Provenance
The change is summarized in firmware 3.2 release notes.
